# Project 1: Sales Data Analysis Dashboard
**Navyan Data Analytics Internship**

## Project Structure
```
├── sales_analysis.py       ← Main Python analysis script
├── sales_dashboard.html    ← Interactive HTML dashboard
├── dashboard_data.json     ← Auto-generated by Python (after running)
├── sales_dashboard.png     ← Auto-generated chart image
└── superstore.csv          ← Your dataset goes here
```

## Setup

### 1. Install dependencies
```bash
pip install pandas matplotlib seaborn openpyxl
```

### 2. Add your dataset
Place your Superstore CSV/Excel file in the same folder.
Update `DATA_PATH` in `sales_analysis.py` to match your filename.

### 3. Run the analysis
```bash
python sales_analysis.py
```

This will:
- Clean and validate your data
- Print KPIs to the terminal
- Generate `sales_dashboard.png` (static chart)
- Export `dashboard_data.json` (for the HTML dashboard)

### 4. View the interactive dashboard
Open `sales_dashboard.html` in your browser.

> **Note:** To load real data into the HTML dashboard, replace the
> `generateSampleRows()` call in the script with:
> ```js
> fetch('dashboard_data.json').then(r => r.json()).then(data => { REAL_DATA = data; applyFilters(); });
> ```
